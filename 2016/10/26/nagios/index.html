nagios

  • 在工作中，尤其是做运维，对服务器的管控，很重要，今天我搭建了服务器监控nagios。工作原理：数据收集–>数据存储–>数据分析–>报警处理–>数据显示

  • 需求：通过nagios对磁盘和web服务器进行监控并且发邮件报警

  • 需求监控30.37的 30.37主机//http:80 端口 OK 5m KO 1m 3次状态一样最终确定服务器

  • 30.37 7*24 监控

  • service 06-24

  • warn | fail | OK unknown ：

  • email ： proo@qq.com

  • notify in 2h 发送 报警的频率

  • time 5*8 上班时间

需要安装软件 nagios nagios-plugins nagios-web

  • 我是在官网找到的rpm包 ，然后通过记录到yum然后配置软件仓库。

    yum install nagios nagios-plugins nagios-web

创建用户

useradd nagios
groupadd nagcmd   
gpasswd -a nagios nagcmd     ---nagios添加到nagcmd组

修改配置文件

  • 添加主机配置文件

    vim /etc/nagios/objects/hosts/30.6.cfg
    define host{

    use                     linux-server
host_name               vm130
alias                   vm130
address                 192.168.32.130
max_check_attempts      3

    }

    define service{

    use                             local-service
host_name                       vm130
service_description             HTTP
check_command                   check_http
notifications_enabled           1

    }
