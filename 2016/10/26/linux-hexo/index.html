linux+hexo

  • 做为一个计算机在运维道路上的人应该有一个自己的博客，我找到了很多开源项目，因为我是学生党，没有钱只能找免费的项目做博客，这里我找到了linux+hexo+github 完全托管搭建自己的博客。

1.首先去github官网去申请自己的账号。

  • github 为用户提供了用户名的二级博客，同时拥有300M的免费空间，写博客完全够用了。

注意：在部署环境的时候需要linux镜像的软件仓库，以及epel项目的仓库。

  • 这里软件yum仓库我就不做介绍了

安装基本的软件，准备环境。

yum -y install gcc zlib-devel openssl-devel perl cpio expat-devel gettext-devel curl autoconf

安装git node

yum istall git nodejs

python必须是2.6以上的版本

安装npm

yum install npm

安装hexo

npm install -g hexo

创建文件blog目录

  • 创建一个blog的工作目录

    mkdir /hexo
    cd /hexo
    hexo init #初始化hexo
    ls # 出现很多文件 即表示成功
    npm install gexo-deployer -git –save #安装依赖

生成静态网页

hexo generate #可以用g表示generate
  • 生成完毕之后，我们可以看到多了一个public文件夹，这就是我们所谓的静态博客的目录，如果我们需要部署到服务器或者托管平台，只要将hexo生成静态之后，将public文件夹里的文件传上去就可以了,其他系统文件还是放在本地。

本地预览

hexo server
  • 浏览器中输入http://localhost:4000 可以在本地查看

配置_config.yml

deploy:
  type:git
  repository: git@github.com:prokao/prokao.github.io.git
  branch:master

*链接github仓库 实现网络同步

hexo g -d
