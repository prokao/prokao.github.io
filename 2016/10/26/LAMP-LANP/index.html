LAMP LANP

  • LAMP linux+apache+mysql+php python nginx

1.nginx安装 yum -y install nginx

在mysql中给app 创建用户 表 授权

2.mysql安装 yum -y install mysql mysql-server mysql-devel

  • 创建用户授权创建表 我在MySQL里面做了介绍，就不在这里阐述了

3.php安装 yum -y install php

4.php-mysql安装 yum -y install php-mysql [安装了什么编程语言，就要安装相对应的（语言-mysql）包 比如：php php-mysql]
5.dns安装 yum -y install bind bind-libs

6.spawn-fcgi安装 需要下载安装包
配置spawn-fcgi

vim /etc/sysconfig/spawn-fcgi
取消下面两行代码的注释，如果第二行代码是-u apache -g apache的话，需要改成-u nginx -g nginx
SOCKET=/var/run/php-fcgi.sock
OPTIONS=”-u nginx -g nginx -s $SOCKET -S -M 0600 -C 32 -F 1 -P /var/run/spawn-fcgi.pid – /usr/bin/php-cgi”
配置nginx 以虚拟主机的方式
vim /etc/nginx/conf.d/www.conf
在配置虚拟主机的基础下加入以下代码
location ~ .php$ {
root /var/www/html/www;
fastcgi_pass unix:/var/run/php-fcgi.sock;
fastcgi_index index.php;
fastcgi_param SCRIPT_FILENAME $document_root$fastcgi_script_name;
include fastcgi_params;
}
