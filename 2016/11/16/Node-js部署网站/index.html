Node.js部署网站

node.js 是一个很好的开源代码，多个平台使用，运用与多个环境。有一些内置模块。
这里我采用linux6.7

安装nodejs

  1. yum install nodejs

查看版本

node -v

运用

node app.js #多数文件需要打开修改监听的端口改为80

后台运行

  1. nohup node app.js & # nodejs 自带的后台运行 但是不支持日志

  2. forever

npm install forever
forever start app.js # 启动
forever stop app.js # 停止
forever start -i forever.log -o out.log -e err.log app.js # 指定日志目录
